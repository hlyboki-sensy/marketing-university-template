# Escalation queue

Питання/рішення, де Claude не може (або не має) приймати рішення сам. Користувач має відповісти або дозволити.

Формат:

```
## <YYYY-MM-DD HH:MM>: коротка суть
Context: що сталося / яке джерело / який claim
Question: конкретне питання до користувача
Status: open | resolved
Resolution: (заповнюється, коли відповідь отримано)
```
